#第七章 系统地讨论类型

对于还未初始化的东西，都会将undefined赋给它。

**undefined和null各自的用途**
在应该提供一个对象，但无法创建或找到时，将提供null——null用于表示对象不存在；  
在变量未初始化、对象没有指定属性或数组没有指定元素时，将返回undefined。

JavaScript灰色地带**NaN**——无法表示的数字(Not a Number)<u>*其类型为number*</u>

NaN不仅用于表示所有无法表示的值还是JavaScript中唯一一个不与自己相等的值！
NaN != NaN

>>	}

>>+ 布尔和字符串的情况
>>
>>将布尔值转换为数字后再进行比较

+ '==='严格相等规则

当且仅当两个值的**类型**和**值**都相同时，他们才是严格相等的

+ 拼接和加法运算符

将数字和字符串相加时，**执行的是拼接**，而**不是**加法运算

+ 如何判断两个对象是否相等
==和===都可用来判断，检查两个对象变量是否相等时，比较的是指向对象的**引用**

别忘了，*对象变量储存的是指向对象的引用，因此比较两个对象变量时，实际上比较的是指向对象的引用。*

仅当两个引用指向的是**同一个**对象时，它们才**相等**

+ truthy & falsey 真值和假值

当用于条件表达式中时，被视为true或false的值成为真值和假值

除五个假值：**undefined、null、0、""、和NaN**外其他的值都是真值。

+ 字符串方法和属性简明教程

1.属性length

	- 指出字符串包含多少个字符，为迭代字符串中的字符提供了极大的便利

2.方法charAt

	- 方法charAt将一个整数作为参数（这个参数必须在0和字符串长度减1之间），并返回一个字符串，其中包含指定位置处的字符。可以认为字符串有点像数组，其中每个字符都有对应的索引，而索引从0开始（同数组一样）。如果指定的索引大于或等于字符串的长度，这个方法将返回一个空字符串。

3.方法indexOf

	- 这个方法将一个字符串作为参数，并在字符串中该参数首次出现的位置返回该参数中第一个字符的索引。

	- 例：
	
	'''
	xxx.indexOf("xxx")	//找到字符串中第一个“xxx”

	xxx.indexOf("xxx"，5)	//从索引5开始查找第一个“xxx”

	//如没有找到指定的字符串，将返回索引-1
	'''

4.substring方法

	- 方法substring将两个索引作为参数，提取并返回这两个索引之间的子串。

	- 例：

	'''
	xxx.substring(5);	//提取从指定索引到字符串末尾的子串。

	xxx.substring(5,10);	//提取并返回从索引5到索引10（不包括）的子串。
	'''

5.方法split

	- 将一个用作分隔符的字符作为参数，并根据这个分隔符将字符串分成多个部分。

	- 例：

	'''
	var data = "name|phone";

	var vals = data.split("|");

	console.log("Split array is ", vals);

	//控制台输出Split array is ["name","phone"]
	'''

第六章**要点**

+ 在JS中，类型分两组：**基本类型**和对象。不属于基本类型的值都是**对象**。

+ 基本类型包括数字、字符串、布尔值、null、undefined;其他的值都是对象。

+ undefined意味着变量（属性或数组元素）还未初始化。

+ null表示“无对象”。

+ NaN指的是“非数字”，但更准确地说，NaN指的是在JS中无法表示的数字。NaN的类型为数字。

+ NaN与包括它自己在内的任何值都不相等，因此要检查一个值是否是NaN，应使用函数isNaN。

+ 要避免进行类型转换，可使用===，但在有些情况下，自动执行类型转换的==可提供极大的便利。

+ 仅当两个变量包含的对象引用指向同一个对象时，它们才相等。
